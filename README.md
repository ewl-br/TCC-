# TCC-
membros:Cassio,Ezequiel,Pedro,Guilherme,Vinícius_e_Eduardo

Sistema inteligente de Cronogramas Escolares, conhecido como "Estuda+", é voltado para estudantes de escolas de ensino médio, com auxílio de uma geração de cronogramas feitas por uma Inteligência Artificial(OpenAI), para estudar para exames como o SSA e o ENEM, além de auxiliar os estudos escolares pessoais do aluno.
