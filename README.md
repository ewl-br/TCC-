Sistema inteligente de Cronogramas Escolares, conhecido como "Estuda+", é voltado para estudantes de escolas de ensino médio, com auxílio de uma geração de cronogramas feitas por uma Inteligência Artificial(OpenAI), para estudar para exames como o SSA e o ENEM, além de auxiliar os estudos escolares pessoais do aluno. O Trabalho de Conclusão de Curso seguiu Metodologias Ágeis, utilizando o Scrum,Trello e Ummense.
